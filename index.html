Subconscious UI

Subconscious is a simple and clean python based desktop UI for interacting with the most popular LLMs. Built with Langchain on the backend it supports out of the box, OpenAI, Google, Anthropic, Ollama and Hugging Face. Currently only built for windows. The code is open sourced, give us a star on github.
